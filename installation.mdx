## Install the package

Install `theutilyes` using your preferred package manager:

<CodeGroup>

```bash npm
npm install theutilyes
```

```bash yarn
yarn add theutilyes
```

```bash pnpm
pnpm add theutilyes
```

</CodeGroup>

## Requirements

`theutilyes` is a TypeScript utility library with minimal dependencies:

- Node.js (recommended: version 14 or higher)
- TypeScript (for TypeScript projects)

<Note>
  The library has zero external dependencies except for built-in Node.js modules (`fs`) and `robert` for HTTP functionality.
</Note>

## Verify Installation

After installation, verify that the package is installed correctly:

```bash
npm list theutilyes
```

You should see the package listed in your dependencies.
