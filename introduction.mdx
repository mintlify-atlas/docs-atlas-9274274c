## Welcome to util

**util** is a collection of lightweight TypeScript utility functions designed to simplify common operations in your projects. From type-safe comparisons to file system helpers and TypeScript type utilities, util provides the building blocks you need without the bloat.

## Quick Example

```typescript
import { isEven, isDirectory, parseEmoji } from 'theutilyes';

// Type-safe number utilities
console.log(isEven(4)); // true

// File system helpers
if (isDirectory('./src')) {
  console.log('Directory exists');
}

// Emoji parsing
const emoji = parseEmoji('<:wave:123456789>');
console.log(emoji); // :wave:
```
