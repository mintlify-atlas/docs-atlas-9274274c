## Number Parity Functions

### isOdd

Check if a number is odd.

```typescript
function isOdd(num: number): boolean
```

#### Parameters

<ParamField path="num" type="number" required>
  The number to check
</ParamField>

#### Returns

`boolean` - Returns `true` if the number is odd, `false` otherwise

#### Examples

```typescript
import { isOdd } from 'util-library';

isOdd(1);   // true
isOdd(2);   // false
isOdd(3);   // true
isOdd(0);   // false
isOdd(-1);  // true
isOdd(-2);  // false
```

## Boolean Conversion Functions

### toBoolean

Convert any value to a boolean using JavaScript's truthiness rules.

```typescript
function toBoolean(item: any): boolean
```

#### Parameters

<ParamField path="item" type="any" required>
  The value to convert to boolean
</ParamField>

#### Returns

`boolean` - The boolean representation of the value

#### Examples

```typescript
import { toBoolean } from 'util-library';

// Truthy values
toBoolean(1);          // true
toBoolean("hello");    // true
toBoolean([]);         // true
toBoolean({});         // true
toBoolean(true);       // true

// Falsy values
toBoolean(0);          // false
toBoolean("");         // false
toBoolean(null);       // false
toBoolean(undefined);  // false
toBoolean(false);      // false
toBoolean(NaN);        // false
```

#### Implementation

```typescript
export function toBoolean(item) {
  return !!item;
}
```

## Special Validators

### isH

Check if the input is the letter 'h' (as a string) or the number 8.

```typescript
async function isH(input: any): Promise<boolean>
```

#### Parameters

<ParamField path="input" type="any" required>
  The value to check
</ParamField>

#### Returns

`Promise<boolean>` - Returns `true` if the input is 'h' or 8, `false` otherwise

#### Examples

```typescript
import { isH } from 'util-library';

// Valid H values
await isH('h');    // true
await isH(8);      // true

// Invalid values
await isH('H');    // false (case-sensitive)
await isH('i');    // false
await isH(7);      // false
await isH(9);      // false
await isH('8');    // false (string number)
```

<Note>
  This is an async function. The reason for the "H" = 8 association is likely a reference to the visual similarity between the letter H and the number 8.
</Note>

#### Implementation

```typescript
export async function isH(input: any) {
  if (typeof input === 'number' && input === 8) return true;
  if (typeof input === 'string' && input === 'h') return true;
  return false;
}
```
